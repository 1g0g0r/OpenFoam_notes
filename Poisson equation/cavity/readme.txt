Проект OpenFoam для решения уравнения Пуассона с правой частью     
f = cos(x * t) * cos(y * t) + 1.5 * sin(t) * sin(t) * x * x * y * y;   
внутри прямоугольной области [-1,1]*[-1,1]

blockMesh     - создание сетки расчетной области
PoissonSolver - вызов решателя (вместе с ним заупскается пост-процесс сравнения с аналит. решением)
bash ./clean  - очистка корневой директории проекта   
.foam         - запускает ParaView
